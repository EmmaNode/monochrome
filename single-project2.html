39andCanine

A collaborative project with peer developers providing an easy tool for users to find the breed of a canine by uploading an image.

View Website

Focus

Machine Learning & Materialize Framework

Date

February, 2018

Agency

General Assembly

39 and Canine is a project that uses Google's Inception v3 model with Tensorflow along with Django as a server to create a web interface to allow users to upload a photo of a dog for breed classification (the model has been trained with ~120 breeds). The web interface is made using Materialize CSS and jQuery It is extension of this project and the Django server is based on this project's, which can be found here on github.
